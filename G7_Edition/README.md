# G7 Edition — The Rollover Reckoning

## Overview
This analysis tracks the maturity gap between G7 nations and breakout emerging markets using the Weighted Average Maturity (WAM) and the Interest-to-Revenue Ratio.

## Data Sources
- **Chart 1 (WAM Comparison):** US Treasury / National Treasury Portals (MX, IN, BR)
- **Chart 2 (Interest Burden):** OECD / IMF Fiscal Monitor

## Environment
Run `conda env create -f environment.yml` to set up the analysis environment.

## Usage
1. Activate the environment: `conda activate g7_edition`
2. Launch Jupyter: `jupyter notebook analysis.ipynb`
